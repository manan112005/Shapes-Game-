# 🎮 Shapes Game

A simple and fun Shapes Game built using **HTML** and **CSS**.  
This project focuses on practicing UI design, layout, and styling.

## 🚀 Features
- Different shapes displayed on the screen
- Interactive hover effects
- Clean and minimal design
- Smooth styling using CSS

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📂 How to Run
1. Download or clone this repository
2. Open the `index.html` file in your browser
3. Start playing!

## ⚠️ Limitations
- No data saving
- No backend functionality
- Works only in the browser
